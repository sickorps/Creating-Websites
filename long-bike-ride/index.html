This page is awesome

The point of this page is to have the nav bar stuck to the top of the page, and have the content scroll over the background image at the top of the page. This page is just a quick example and can be taken further. The point is, this works using positioning. The nav is positioned fixed in order to stay at the top. You could also do position:absolute and have set top: 0, however, that will interfere with the Flexbox container, so we used fixed instead. The background image is also position: fixed in order to keep it at the top of the page. The content of the page is position: absolute with a little bit of top in order to allow it to smoothly pass over the background image. We simply top just shy of 10px less than the height of the image itself. The footer is also contained by the element containing the body text of the page. This allows the content text and the footer to be displayed block without interrupting the flow of the position: absolute element we used to keep the body text all in order and scrolling at the same space. Otherwise we would end up with something kinda like parallax.
